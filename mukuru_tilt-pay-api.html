Crypto infrastructure for African payments

Tilt Pay API connects Mukuru to global crypto rails — stablecoin settlement, JIT remittance, crypto-backed cards and hard-currency virtual accounts — all without requiring Mukuru to build any of this themselves.

⚠
Future roadmap — not all capabilities are live today
This page describes Tilt Pay API's planned crypto infrastructure. Ripple and Fireblocks integrations are active today; Stellar, Circle Mint/Redeem, Bridge crypto cards and virtual accounts are planned capabilities on the roadmap.

Detailed capabilities

Just-in-time remittance settlement

Sending partners fund payments using crypto instead of pre-funding local accounts. Tilt Pay API converts and disburses local currency the moment the transaction is initiated.

Sending partner (USD/EUR)→ USDC / XRP→ Tilt Pay API→ Local currency→ Recipient
No pre-funded accounts in receiving countries — capital efficiency for remittance partners
Settlement in seconds vs. days on traditional correspondent banking rails
Works across all 9 Tilt Pay API markets from a single crypto funding source
Ripple ODL and Stellar provide the fastest crypto settlement rails for this use case
♾
Stablecoin Mint & Redeem

A Mint & Redeem agreement with Circle means Mukuru can convert between USD fiat and USDC at 1:1 with no spread — eliminating FX exposure when holding stablecoins.

USD fiat→ Mint (Circle)→ Tilt Pay API⇄ USDC⇄ Tilt Pay API→ Redeem (Circle)→ USD fiat
1 USDC = 1 USD always — no market spread, no exchange rate risk on stablecoin holdings
Mukuru can be long in USDC without losing money vs. holding USD fiat
Instant liquidity management — mint when you need USDC, redeem when you need fiat
Tilt Pay API integrates directly with Circle's API for seamless mint/redeem operations
💳
Crypto-backed cards via Bridge

Bridge issues Visa/Mastercard-branded cards backed by stablecoin balances. Unlike traditional cards, these are not restricted by cross-border card routing rules — they work anywhere in the world.

Mukuru Wallet (USDC)→ Tilt Pay API→ Bridge card→ Visa / Mastercard→ Any merchant globally
Cards linked to USDC balance in Mukuru wallet — no bank account required
No traditional cross-border card restrictions — no BIN routing limits, no issuer country blocks
Settlement happens in stablecoin rails, bypassing correspondent banking entirely
Tilt Pay API connects the card to Mukuru's wallet infrastructure — single integration
🏠
USD / EUR / GBP virtual accounts

Companies and individuals can receive hard-currency payments via real-looking account numbers (US routing, IBAN, UK sort code) without needing a traditional bank account — settled back to USDC via Tilt Pay API.

USD / EUR / GBP sender→ Virtual IBAN / ACH→ USDC→ Tilt Pay API→ Mukuru Wallet
African businesses can invoice internationally in USD, EUR or GBP without a US/EU/UK bank
Diaspora senders can pay into a familiar bank account number that routes to Mukuru wallet
Received funds auto-settle to USDC and connect back through Tilt Pay API to Mukuru
No correspondent banking — faster, cheaper and more accessible for underserved markets
⚡
All of this via one Tilt Pay API integration — zero additional development for Mukuru

Mukuru benefits from every one of these capabilities through their existing Tilt integration — without navigating technical complexity, or building separate infrastructure. One API.
